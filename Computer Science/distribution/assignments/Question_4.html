4 Question 4

Language: Beginning Student

Preparation: Module 4, through slide 11

The set of numbers \(a+c\sqrt{b}\), where \(b\) is a natural number greater than 1 without repeated prime factors, and \(a\) and \(c\) are rational numbers, forms a quadratic field. Such numbers can be represented exactly in Racket (without using inexact numbers or the sqrt function) as (make-qfe a c b) using the following definition:

(define-struct qfe (rat irrat base))

Write the function add which consumes two qfe structs and produces the qfe struct representing their sum. Also write sub, mult, and div which implement subtraction, multiplication, and division.

As an example, the number \(1-\sqrt{5}\) is represented by (make-qfe 1 -1 5), and the result of multiplying this number by itself, \(6-2\sqrt{5}\), is represented by (make-qfe 6 -2 5).

You should catch division by zero and signal an error using the predefined error function (look this function up in the Racket documentation to learn how to use it), using an error message that contains the word "zero". Each of the functions should signal an error if the two arguments have different bases, using an error message that contains the word "different". You do not need to check that the base has no repeated prime factors (but this is an interesting question to think about).
