15 Question 15

Language: Intermediate Student with Lambda.

Preparation: Module 6

Using abstract list functions, it is possible to implement many other functions in a single definition without helper functions and without explicit recursion – that is, without the name of the function appearing in the body of its definition. The abstract list functions, of course, use recursion to get the work done. Here’s an example we saw on the lecture slides:

(define (sum-list lst)
  (foldr + 0 lst))

Use the predefined abstract list functions map, filter, and foldr to implement unique-left and unique-right from Question 9, but without helper functions or explicit recursion. You can and should use lambda.

As in Question 9, please do not use the built-in reverse or remove functions. If you did not use pure structural recursion in Question 9, you will find this task easier if you go back and fix Question 9 first.
